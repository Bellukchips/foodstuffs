
## Create Database

```bash
foodstuffs
```

## Migrate All Table

```bash
php artisan migrate
```

## Use Database Seeder For Admin

```bash
php artisan db:seed
```

## Create file .env

```bash
create file .env and copy all content in file .env.example
```

## Generate Key

```bash
php artisan key:generate
```

## Running

```bash
php artisan serve
```

## Another way to run the app

launch url

```bash
    foodstuffs.test
```

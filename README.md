## Run Composer update / install
```bash
composer update
```
## Create Database

```bash
foodstuffs
```

## Migrate All Table

```bash
php artisan migrate
```

## Use Database Seeder For Admin

```bash
php artisan db:seed
```

## Create file .env

```bash
create file .env and copy all content in file .env.example
```

## Generate Key

```bash
php artisan key:generate
```

## Running

```bash
php artisan serve
```
